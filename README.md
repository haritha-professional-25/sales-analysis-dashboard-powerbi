
# Sales Analysis Dashboard - Power BI

## Overview
This project is a Power BI Sales Analysis Dashboard created to analyze business performance using sales data.  
The dashboard provides insights into sales, profit, regional performance, yearly trends, and category-wise analysis.

It is designed to help businesses make data-driven decisions by visualizing key metrics in an interactive and user-friendly format.

---

## Dashboard Features

### Key Metrics
- Total Sales
- Total Profit
- Total Categories
- Total Regions
- Sub-Category Count

### Visualizations
- Sales and Target Gauge Chart
- Profit by Region (Donut Chart)
- Sales by Year for Each Region (Line Chart)
- Sales by Year Trend Analysis
- Category Filter/Slicer
- Sub-Category Sales Table by Region

---

## Tools & Technologies
- Power BI
- DAX
- Power Query
- Data Visualization Techniques

---

## Dataset Information
The dashboard uses sales transaction data containing:
- Sales
- Profit
- Region
- Category
- Sub-Category
- Yearly Sales Data

## Insights Generated
- Identify top-performing regions
- Analyze yearly sales growth
- Compare category-wise sales performance
- Track profit contribution by region
- Monitor overall business performance
